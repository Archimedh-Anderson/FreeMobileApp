================================================================================
SECURITY RECOMMENDATIONS
================================================================================

1. DO NOT commit .env file to GitHub
2. CHANGE JWT_SECRET_KEY in production secrets
3. Use Streamlit Cloud secrets for sensitive data
4. Rotate API keys periodically
5. Monitor API usage and costs

================================================================================
TROUBLESHOOTING
================================================================================

Issue: App crashes on startup
Solution: Check logs in Streamlit Cloud dashboard

Issue: Gemini API not working
Solution: Verify API key in secrets configuration

Issue: Missing dependencies
Solution: Check requirements.txt is up to date

Issue: File upload fails
Solution: Check file size limits (max 200MB configured)

================================================================================
POST-DEPLOYMENT VERIFICATION
================================================================================

Test the following after deployment:

1. App loads successfully
2. Provider selection shows Gemini as available
3. File upload works
4. Classification runs without errors
5. Results display correctly
6. Export functionality works
